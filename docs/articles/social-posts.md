## Short English Post

Most agent demos can call tools. Production agents need harder answers:

Did the tool actually run? Can the task retry safely after a crash? Can we prove what happened later?

AgentLedger is a runtime reliability layer for AI agents. It adds Tool Ledger, checkpoints, evidence, replay, policy/approval/sandbox boundaries, and failure attribution around tool/model execution.

Start with the 3-minute demo: an agent sends an email, the worker crashes before checkpoint, retry does not duplicate the side effect, and Inspector shows the evidence.

GitHub: https://github.com/yaogdu/AgentLedger
